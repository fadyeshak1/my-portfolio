Overview

About the Project

Metro Misr is designed to simplify daily commuting for Cairo Metro users. The app provides accurate station data for all metro lines, allows users to plan trips between any two stations, and uses GPS to detect the nearest station — all in a fast, lightweight package.

The app prioritizes speed and usability, with a clean and intuitive UI that works efficiently on most Android and iOS devices. Dark mode ensures comfortable use at any time of day, and the lightweight architecture ensures fast load times even on lower-end devices.

Trip Planning

Select any two stations and instantly get the number of stops and the route path between them.

GPS Detection

Uses device location services to automatically find and display the nearest metro station.

Design Focus

Clean, fast UI with dark mode support — optimized for quick daily use during commutes.
